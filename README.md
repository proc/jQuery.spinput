# jQuery.spinput

## Description

An ultra-minimalistic jQuery spinner plugin driven by keybindings.

### Usage
	$('input.spinput').spinput({
		step: 1,
		max: null,
		min: 0
	});
#### Options

**step**: number to increment or decrement by (*default: 1*)

**max**: upper bound (*default: none*)

**min**: lower bound (*default: none*)
